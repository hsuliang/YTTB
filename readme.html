關於本工具

這是一個專為內容創作者打造的 AI 生產力中心。透過 Google Gemini 的強大能力，將您的影片或語音逐字稿（SRT）一鍵轉化為部落格文章、社群貼文與電子報。

無論您是 YouTuber、Podcaster 還是講師，這個工具都能幫您節省 90% 的內容再製時間，讓創作效率最大化！

四大核心功能

1

智慧字幕整理 (Tab 1)

  • 內建 DownSub 字幕下載教學
  • 自動合併短行、拆分長句
  • 時間軸錯誤修復
  • 批次取代錯別字
  • 生成後提供詳細整理報告
2

部落格文章生成 (Tab 2)

  • AI 文本優化 (防呆機制保護)
  • 自訂寫作人稱與語氣
  • SEO 標題與關鍵字建議
  • Quill 編輯器即時預覽
3

社群貼文生成 (Tab 3)

  • Facebook / IG / Line 多版本
  • 自動生成 Hashtags
  • 客製化 CTA (行動呼籲)
4

電子報與其他 (Tab 4+)

  • 電子報內容生成
  • 深色模式 (Dark Mode)
  • API Key 申請詳細教學

快速上手指南

  1. 步驟一：設定 API Key

    點擊右上角鑰匙圖示，展開「📖 如何免費申請 API Key？」教學，跟著步驟免費取得金鑰並儲存。

  2. 步驟二：匯入與整理字幕

    在 Tab 1 點擊「如何取得字幕？」依照教學下載 SRT 檔。貼上後設定參數，點擊「開始整理」，系統會彈出整理報告並引導您下一步。

  3. 步驟三：生成內容

    前往 Tab 2, 3, 4。建議先在 Tab 2 使用「優化文本」提升品質。設定好您想要的風格後，點擊生成按鈕，AI 就會為您產出專業內容。
