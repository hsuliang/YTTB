# ㄚ亮笑長的內容創作助手 AI

這是一個專為內容創作者打造的全方位 AI 助手，能夠幫助您將字幕檔案（SRT）快速轉化為高品質的部落格文章、社群貼文以及電子報內容。結合了最新的 Google Gemini AI 技術，讓您的內容產出效率翻倍！

![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ 主要功能

### 1. 智慧字幕整理 (Tab 1)
*   **直覺匯入介面**：全新的空狀態設計，支援貼上文字或拖曳 `.srt` / `.txt` 檔案。
*   **新手友善教學**：內建折疊式教學，手把手教您如何使用 DownSub 或 Faster Whisper 取得影片字幕。
*   **結構優化**：自動合併過短的字幕行、拆分過長句子。
*   **時間軸修復**：智慧修正重疊或間隔過大的時間戳。
*   **批次取代**：可設定規則快速修正錯別字或專有名詞，並支援儲存常用規則範本。
*   **視覺化報告**：整理完成後，自動彈出詳細數據報告，並引導下一步。
*   **AI 輔助**：自動生成影片章節與摘要。

### 2. 部落格文章生成 (Tab 2)
*   **AI 文本優化**：一鍵將口語化的逐字稿轉化為流暢的文章底稿。
*   **風格變體中心 (Variation Hub)**：
    *   **多樣化風格**：內建「親切對話」、「專家顧問」、「情境帶入」等多種專業寫作風格。
    *   **自訂指令**：可輸入任意 Prompt 來微調文章語氣。
    *   **範本管理**：將您調教好的風格指令儲存為範本，隨時取用。
    *   **精準控制**：支援「覆蓋主畫面設定」選項，讓您完全掌控 AI 的寫作方向。
*   **寫作風格精靈**：可自訂人稱、語氣與字數。
*   **SEO 建議**：自動生成 SEO 標題、描述、關鍵字與內部連結建議。
*   **即時預覽與編輯**：內建 Quill 編輯器，支援 HTML 與 Markdown 匯出。

### 3. 社群貼文生成 (Tab 3)
*   **一鍵多平台**：一次生成 Facebook、Instagram、Line 三種不同風格的貼文。
*   **社群風格精靈**：針對各平台特性（如 IG 的 Hashtag、Line 的親切語氣）進行客製化。
*   **進階風格變體**：透過 Variation Hub 生成「視覺系排版」、「懸疑提問」、「懶人包」等吸睛貼文。
*   **自動 Hashtag**：AI 自動分析內容並推薦熱門標籤。

### 4. 電子報內容生成 (Tab 4)
*   **受眾客製化**：針對不同訂閱者（如忠實粉絲、潛在客戶）調整語氣。
*   **電子報風格變體**：快速生成「粉絲專屬信」、「商務提案」、「故事行銷」等不同類型的郵件內容。
*   **CTA 優化**：自動生成具備高轉換率的行動呼籲。

### 5. 系統與設定
*   **API Key 申請教學**：內建詳細圖文步驟，教您免費申請 Google Gemini API Key。
*   **隱私安全**：API Key 僅暫存於瀏覽器，絕不傳送至第三方伺服器。
*   **外觀自訂**：支援深色模式 (Dark Mode) 與多種配色主題。
*   **草稿自動儲存**：防止意外關閉導致內容遺失。

## 🚀 如何開始

1.  **開啟程式**：直接用瀏覽器打開 `index.html`。
2.  **設定 API Key**：點擊右上角鑰匙圖示，依教學申請並輸入 Google Gemini API Key。
3.  **匯入字幕**：在 Tab 1 貼上字幕內容。
4.  **開始創作**：依照分頁順序，享受 AI 帶來的極速創作體驗！

## 🛠️ 技術堆疊

*   **Frontend**: HTML5, CSS3 (Tailwind CSS), JavaScript (Vanilla ES6+)
*   **AI Model**: Google Gemini 1.5 Flash (via Google Generative AI SDK)
*   **Editor**: Quill.js
*   **Icons**: Heroicons
