# ㄚ亮笑長的 Youtube 工具一條龍

歡迎使用「ㄚ亮笑長的 Youtube 工具一條龍」！這是一個專為 YouTube 內容創作者、Podcaster 和教育工作者設計的全方位內容再利用 (Repurpose) 工具。

您的影片/音訊逐字稿 (SRT 檔案) 只需要上傳一次，就能自動轉換為部落格文章、社群貼文、影片摘要與章節。本工具串接 Google Gemini AI，讓您的內容創作流程「一條龍」搞定！

## ✨ 核心功能

本工具分為三大模組，從原始字幕到最終產出，提供全方位的 AI 輔助。

### 1. 字幕整理與優化 (Tab 1)

* **智慧匯入**：支援貼上 SRT 內容、拖曳 `.srt`/`.txt` 檔案，或點擊上傳。
* **字幕清理**：
    * 自訂單行字數 (10-50字)。
    * 自動合併過短行 (例如 3 字以下)。
    * 一鍵移除所有標點符號。
* **時間軸校正**：
    * 修復時間戳重疊或過短的間隔 (例如 100ms 內)。
    * 支援全體時間軸平移 (毫秒級)。
* **批次取代**：設定多組關鍵字，在整理時一次全部取代 (例如： "阿亮" -> "ㄚ亮")。
* **AI 輔助** (需 API Key)：
    * **生成影片章節**：自動生成 YouTube 格式的時間戳章節。
    * **生成影片摘要**：自動撰寫 150 字的影片簡介。
* **匯出**：一鍵匯出清理完成的 `.srt` 檔案。

### 2. 部落格文章生成 (Tab 2)

* **AI 文本優化**：一鍵將口語化的逐字稿優化為流暢、易讀的書面文章。
* **AI 文章生成**：
    * 串接 Gemini AI，將文本轉換為結構完整的 HTML 部落格文章。
    * 自動插入影片標題 (H1) 與 YouTube 嵌入框。
    * **AI 寫作風格精靈**：透過引導式問答，客製化 AI 的寫作風格 (例如：開頭摘要、小標題風格、加入粗體、金句、表格等)。
* **SEO 整合**：
    * AI 自動生成 SEO 標題、搜尋描述、固定網址與標籤。
    * **進階 SEO 分析**：AI 協助分析核心關鍵字、長尾關鍵字，並提供內部連結建議。
* **多版本生成**：可多次生成不同風格的文章版本並隨時切換。
* **匯出**：支援下載 `.html` 與 `.md` (Markdown) 檔案。

### 3. 社群貼文生成 (Tab 3)

* **多平台一次搞定**：AI 一次生成 Facebook, Instagram, Line 三種平台的專屬貼文。
* **AI 社群風格精靈**：客製化各平台的貼文偏好 (例如：FB 條列化、IG 加強 Emoji、Line 口語化)。
* **多版本生成**：可快速生成多種不同風格的社群貼文，並在版本間切換。
* **一鍵複製**：快速切換平台並複製文案。

### 🌟 全域功能

* **主題切換**：內建多種亮色與暗色佈景主題。
* **狀態保存**：所有草稿 (文章、貼文、設定) 都會自動暫存在瀏覽器 (`localStorage`) 中，刷新頁面也不會遺失。
* **API Key 安全性**：Gemini API Key 僅儲存於瀏覽器的 `sessionStorage`，2 小時後或關閉瀏覽器即自動清除。

## 🛠️ 使用技術

* **前端**：HTML5, CSS3, Vanilla JavaScript (ES6+)
* **CSS 框架**：[Tailwind CSS](https://tailwindcss.com/)
* **AI 核心**：[Google Gemini API](https://aistudio.google.com/)
* **核心邏輯**：
    * `srt-processor.js`: 負責 SRT 字幕的解析、處理與字串化。
    * `gemini-api.js`: 封裝 Gemini API 呼叫，包含自動重試與錯誤處理。
    * `tab1.js`, `tab2.js`, `tab3.js`: 各分頁的獨立業務邏輯。
    * `state.js` & `app.js`: 全域狀態管理與應用程式初始化。

## 📖 如何使用

### 步驟一：設定 API Key (必要)

1.  點擊頁面右上角的「**鑰匙** 🔑」圖示。
2.  前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 申請您的 Gemini API Key。
3.  將您的 API Key 貼入輸入框中，點擊「**儲存金鑰**」。
4.  儲存成功後，狀態會顯示「已設定」，所有 AI 相關按鈕將會啟用。

### 步驟二：上傳字幕 (Tab 1)

1.  在「**1. 字幕整理與優化**」分頁。
2.  將您的 `.srt` 檔案拖曳至虛線框內，或貼上字幕內容。
3.  (可選) 調整「單行字數」、「合併字數」等設定。
4.  (可選) 點擊「**批次取代**」設定您常用的修正規則。
5.  點擊「🚀 **開始整理**」。
6.  (可選) 點擊「**生成章節**」或「**生成摘要**」取得 AI 輔助內容。
7.  (可選) 點擊「**匯出 SRT 檔案**」下載清理過的字幕。

### 步驟三：生成部落格文章 (Tab 2)

1.  切換至「**2. 部落格文章生成**」分頁。
2.  (強烈建議) 點擊「**優化文本 (建議)**」按鈕，讓 AI 先將逐字稿轉為流暢文章。
3.  (可選) 點擊右上角「**齒輪** ⚙️」圖示，使用「**AI 寫作風格客製化精靈**」調整風格。
4.  填寫「文章標題」、「YouTube 影片 ID」。
5.  點擊「**生成部落格文章**」。
6.  成果將顯示在右側。您可以在「文章預覽」、「HTML 原始碼」、「SEO 建議」等視圖中切換。
7.  (可選) 點擊「**另一版本**」來生成不同風格的內容。

### 步驟四：生成社群貼文 (Tab 3)

1.  切換至「**3. 社群貼文生成**」分頁。
2.  (可選) 點擊「**前往優化 >**」使用 Tab 2 的優化文本。
3.  設定「貼文目標」、「長度」、「語氣」等。
4.  (可選) 點擊右上角「**齒輪** ⚙️」圖示，客製化社群風格。
5.  點擊「**生成社群貼文**」。
6.  在 Facebook, Instagram, Line 之間切換預覽，並可點擊「**複製內容**」。
